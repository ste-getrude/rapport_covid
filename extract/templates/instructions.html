Aide

La mise a jour des données se fait a partir de l'onglet Mise a jour donné et consiste à téléverser un fichier excel au serveur.

  • Le fichier est une extraction du systeme de l'école
  • L'extention (format) du fichier doit être .xlsx et non pas .xls autrement le processus de mise a jour échouera.
  • Il est impératif que le fichier ait le format suivant (ordre des colonnes):
  • Chaque caractéristique (colone) de l'édudiant doit être inscrite au fichier, dans le cas contraire les données de cet étudiant ne seront pas enregistrées dans le systeme.
  • Si une donné est manquante, un message sera retourné, spécifiant la ou les lignes pour lesquelles il manque de l'informations.
  • Une vérification montre qu'à la ligne 52 le courriel n'est pas au dossier.
  • et que la ligne 478 est vide mais formatée.
  • Les autres données sont tout de même dans le systeme
  • A chaque téléversement, les données déjà dans le systeme sont complètement éffacées et remplacées, ce qui implique que pour avoir l'intégralité des données prise en compte, chaque téléversement doit être fait avec un fichier complet
